Setup git to use ssh key:
    eval "$(ssh-agent -s)"
    ssh-add ~/.ssh/github_rsa


SSH bash settings:
    https://github.com/Russell91/sshrc/
    .sshrc.d/ is filled with symlinks to: ~/.bash_profile, ~/.bash_aliases, ~/.inputrc

Install sshrc configs:
    cp -R --preserve=links  .sshrc.d/ ~/
    cp .sshrc ~/

Fonts:
    Monospace: fonts-inconsolata
